Delivery status/Shipping status of sale and purchase


- Added field named delivery status inside sale order and Shipping Status inside purchase order.

- If all the deliveries of sale order are done, delivery status will be done otherwise delivery status will be open.

- If all the shipments of purchase order are done, Shipping status will be done otherwise shipping status will be open.

- Also can perform analysis based on delivery status/shipping status in sale/purchase Analysis report

- Sales Analysis report
- Purchase Analysis report
